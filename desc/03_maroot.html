마루티

프로젝트 개요 : 웹 기반 온라인 주문 시스템

프로젝트 소속 : HUHS(한양대학교 동아리)

개발기간 : 4개월

프로젝트 참여인원 : 5인

개발 스택

프로젝트를 수행하면서 사용한 개발 기술 요약 정보

  • Nodejs
  • MongoDB
  • AngularJS
  • AWS Cloud Service
  • Javascript

핵심 구현요소

프로젝트의 상세 구현 요소 설명

온라인 주문 기능 구현

사용자가 앱, 웹에서 주문하게 되면 관리용 앱에 해당 주문에 대한 알람이오고 사용자에게 해당 주문이 처리되도록 기능 개발

MEAN Stack 활용 개발

Mongodb, Express, Angularjs, Nodejs 프레임워크를 사용하여 개발

관리용 웹 애플리케이션 개발

웹, 앱으로 관리자가 주문에 대하여 처리하고 사용자의 주문이 알람이 오도록 개발

모바일에서 사용가능한 호환성 고려한 개발

관리자가 모바일로도 상품 등록, 수정, 주문처리가 가능하도록 호환성을 고려하여 개발됨
