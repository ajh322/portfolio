프로젝트 개요 : 차량 가상 튜닝 플랫폼

프로젝트 소속 : 인디카

개발기간 : 1년

프로젝트 참여인원 : 4인

특이사항 : 클라이언트, 백엔드 개발 부분 전부 진행

개발 스택

프로젝트를 수행하면서 사용한 개발 기술 요약 정보

  • 유니티 클라이언트 3D
  • C#
  • Nodejs
  • MongoDB

핵심 구현요소

프로젝트의 상세 구현 요소 설명

실시간 차량 튜닝 어플리케이션

12종의 차량을 실제로 존재하는 튜닝 부품을 직접 장착, 도색이 가능한 실시간 렌더링 어플리케이션

최적화 및 시각성 고려

모바일 디바이스에서 문제되지 않도록 최적화를 고려하며 실제 차량과 유사한 시각적 효과를 위하여 다양한 쉐이더 사용

차량 에셋번들 서버 개발, 연동

차량 모델에 대한 파일을 에셋번들 서버를 개발하여 연동하여 차량을 불러올때 동적으로 다운받도록 처리

차량 메타 데이터 실시간 서버 연동

차량의 썸내일 이미지, 바퀴의 좌표, 바퀴 모델 등 다양한 정보를 실시간으로 관리 가능하도록 개발
