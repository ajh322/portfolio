  • Nodejs
  • MongoDB

  • AWS Cloud Service
  • Javascript

핵심 구현요소

프로젝트의 상세 구현 요소 설명

자체 독립몰 시스템

일반적인 고도몰 기능을 계승하며 스마트스토어 시스템을 기반으로 만든 자체 독립몰 솔루션입니다.

상품 주문, 결제, 배송 시스템

상품에 대한 주문, 결제, 배송 단계에 대한 처리와 사용자에게 알림 메시지와 배송 업체 API 연동을 통한 자동 배송 현황 추적까지 전부 지원합니다.

결제처리

신용카드, 무통장입금, 카카오페이 결제 지원합니다.

회원가입

일반 회원가입, SNS 회원가입, 휴대폰 인증을 지원합니다.

통계 시스템 지원

쇼핑몰의 전반적인 데이터에 대한 그래프화, 판매현황을 한번에 볼 수 있는 대시보드 기능을 지원합니다.

고도화 상품 등록 시스템

상품 등록 단계에서는 본문 에디터, 이미지 업로드, 재고관리, 배송정보 설정 종합적인 기능을 지원합니다.

회원 종합 관리 시스템

회원에 대한 정보 관리와 조회에 대한 기능이 구현되어 있습니다.

주문 종합 관리 시스템

주문에 대한 상태 변경과 각 상황별 예외처리가 구현되어 있습니다.
