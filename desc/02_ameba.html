프로젝트 개요 : 모바일 게임 자동 출시 프로그램

프로젝트 소속 : 개인

개발기간 : 6개월

프로젝트 참여인원 : 1인

개발 스택

프로젝트를 수행하면서 사용한 개발 기술 요약 정보

  • 유니티 클라이언트 3D
  • C#
  • 파이썬
  • 애드몹

핵심 구현요소

프로젝트의 상세 구현 요소 설명

게임 자동 출시 시스템 구현

자동으로 제목과 내부 리소스를 전부 교체하며 15분에 1개의 게임의 빌드부터 출시까지 자동으로 해주는 기능 구현

플레이스토어 등록, 출시 구현

플레이스토어에 APK 업로드, 스크린샷 이미지, 연령조사 최소 입력사항 전부 자동으로 입력되도록 구현

최신 트렌드 리소스 교체 기능

구글 트렌드를 사용하여 최신 트렌드의 이미지를 가져와서 자동으로 내부 리소스와 키워드를 교체해주는 기능 구현

유니티 리소스 교체, 자동 빌드 기능 구현

생성된 프로젝트별로 유니티의 리소스를 자동으로 교체, APK 빌드까지 자동으로 관리하는 빌드 파이프라인 구축

앱 스크린샷 자동 생성

빌드된 APK를 디바이스에 설치하고 인게임 스크린샷을 가져와 파이썬 이미지 라이브러리를 사용하여 플레이스토어 스크린샷용 이미지를 자동으로 만들어주는 기능 구현

크로스프로모션 기능

출시된 앱끼리 자동으로 크로스 프로모션을 하도록 출시한 앱을 등록하고 서로 노출시키는 자동화 시스템 구현

실제 수익 창출

실제로 플레이하는 유저와 광고로 인한 수익이 발생함
