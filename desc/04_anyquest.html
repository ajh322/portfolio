애니퀘스트

프로젝트 개요 : 부모 아이간 숙제를 주는 플랫폼

프로젝트 소속 : 외주

개발기간 : 2개월

프로젝트 참여인원 : 3인

개발 스택

프로젝트를 수행하면서 사용한 개발 기술 요약 정보

  • Nodejs
  • Mysql
  • Android
  • AWS Cloud Service
  • Javascript

핵심 구현요소

프로젝트의 상세 구현 요소 설명

회원가입

네이버, 카카오 SNS 회원가입 기능 구현

프로필 기능 지원

기본적인 프로필 설정, 공지사항 게시판 기능 구현

관리자, 사용자 관계 구현

부모가 아이에게 퀘스트를 주는 앱으로 사용자간 관계를 맺을 수 있도록 기능 구현

퀘스트 기능 구현

관리자가 사용자에게 다양한 퀘스트와 그에 대한 보상을 지불 할 수 있도록 기능 구현
