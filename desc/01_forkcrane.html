포크레인

프로젝트 개요 : 해외 스토어 데이터 크롤링 서비스

프로젝트 소속 : 반골소프트

개발기간 : 1년

프로젝트 참여인원 : 3인

특이사항 : 크롤링, 코어 부분 전부 개발

  • 솔루션 링크

개발 스택

프로젝트를 수행하면서 사용한 개발 기술 요약 정보

  • Nodejs
  • Mysql
  • AWS Cloud Service
  • 크롤링 기술
  • 원가 절감을 위한 최적화

핵심 구현요소

프로젝트의 상세 구현 요소 설명

타오바오 상품 크롤링

타오바오 사이트 상품의 대규모 단위 데이터의 옵션, 가격, 이미지 데이터를 전부 가져와서 국내 스토어에 일괄등록 해주는 기능 구현

스마트스토어 데이터 삽입

크롤링 된 데이터를 스마트스토어 데이터 기준에 맞춰서 삽입하는 기능 구현

쿠팡 데이터 삽입

크롤링 된 데이터를 쿠팡 데이터 기준에 맞춰서 삽입하는 기능 구현

번역처리

상품의 제목, 옵션, 본문의 중국어를 한국어로 번역하는 기능 구현, 최적화를 통하여 번역비 사용비를 대폭 절감함

이미지 처리

상품의 썸내일 이미지, 옵션 이미지를 다운받아서 상품 페이지의 적재적소 위치에 삽입하도록 구현

매출 발생

해당 솔루션을 사용하여 스마트스토어 파워등급 달성, 쿠팡 상반기 매출 4천만원 이상의 매출 발생
